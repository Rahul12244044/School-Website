# School Website Backend - Strapi CMS
A fully-featured headless CMS backend for a school website built with Strapi. This backend provides a complete content management system for events, news articles, photo galleries.


# Installation
1) Install dependencies
npm install 

2) Start the development server
npm run develop 

3) Access the admin panel
http://localhost:1337/admin
