# School Website - React Frontend with Strapi CMS
A modern, responsive school website frontend built with React, Redux Toolkit, and Tailwind CSS. This application connects to a Strapi CMS backend to dynamically manage and display events, news, and gallery content.

#  Features
1) Dynamic Content Management: All events, news, and gallery content loaded from Strapi CMS.
2) Complete Redux State Management: Events, News, Gallery, and UI slices with async thunks.
3) Responsive Design: Mobile-first approach with Tailwind CSS.
4) Image Gallery: Lightbox functionality with navigation and filtering.
5) Search & Filters: Real-time search and category filtering for events and news.
6) Performance Optimized: Code splitting, lazy loading, and optimized bundles.
7) Fully Responsive: Works seamlessly on mobile, tablet, and desktop.
8) Modern UI: Clean design with hover effects, animations, and transitions.

# Tech Stack
1) React 18 - UI Library with Hooks and Functional Components.
2) Redux Toolkit - State Management with async thunks and slices.
3) Tailwind CSS - Utility-first CSS framework for styling.
4) React Router DOM v6 - Client-side routing and navigation.
5) Axios - Promise-based HTTP client for API calls.
6) Lucide React - Beautiful & consistent icon toolkit.
7) date-fns - Modern JavaScript date utility library.
